# ATtiny45 + MCP4151 Digital potentiometer

## Overview

|ATtiny45    |MCP4151        |
|------------|---------------|
|PB0 (pin 5) |SDI/SDO (pin 3)|
|PB1 (pin 6) |CS (pin 1)     |
|PB2 (pin 7) |SCK (pin 2)    |

|ATtiny45 |USBASP |
|---------|-------|
|pin 5    |MOSI   |
|pin 6    |MISO   |
|pin 7    |SCK    |
|pin 1    |RESET  |
